Q: How does it work?

The site is a very basic form, that allows you to login with Twitter to save promises you make to teach. Each of the lessons linked should take an hour or less to teach to someone. The goal is to start collecting feedback about lessons, see if we can get people to try to teach things, and then collect their experiences. Then, I'd like to create repos of lessons that we can fork, send pull requests to, and collaborate on together.
